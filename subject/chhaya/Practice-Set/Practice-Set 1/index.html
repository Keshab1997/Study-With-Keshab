# নাম মোট স্কোর বিস্তারিত

আমার রেজাল্ট

এই অধ্যায়ে আপনার প্রতিটি কুইজের পারফরম্যান্স দেখুন এবং বন্ধুদের সাথে শেয়ার করুন।

আপনি এখনও এই অধ্যায়ের কোনো কুইজে অংশ নেননি।

এখনই কুইজ দিন!

🌟 গুরুত্বপূর্ণ তথ্য ও ধারণা (Practice Set 1)

  • ইতিহাস: কোল বিদ্রোহ (১৮৩১-৩২), সাংবিধানিক সভার প্রথম সভা (9 ডিসেম্বর, 1946), বরদৌলি সত্যাগ্রহ (1928)।
  • বিজ্ঞান (পদার্থ):
    • আলো: লম্ব আপতনে প্রতিসরণ কোণ 0°, গোলীয় দর্পণের ফোকাস দৈর্ঘ্য f = R/2।
    • পদার্থের ধর্ম: আর্কিমিডিসের নীতি (ল্যাকটোমিটার), রোধাঙ্ক (R ∝ ρ)।
    • তড়িৎ-চুম্বকত্ব: সলিনয়েডের কেন্দ্রে সুষম চৌম্বক ক্ষেত্র, কমিউটেটরের ব্যবহার (DC মোটর ও জেনারেটর)।
  • বিজ্ঞান (রসায়ন):
    • কার্বনের ইলেকট্রন বিন্যাস (2, 4), কার্বন ডাইঅক্সাইডে ভরের অনুপাত (3:8), অক্সালিক অ্যাসিড (HOOC-COOH)।
  • বিজ্ঞান (জীববিজ্ঞান): ভিটামিন-A (রেটিনল), SA নোড (পেসমেকার), জাইলেম প্যারেনকাইমা (খাদ্য সঞ্চয়)।
  • ভূগোল: বেঙ্গুয়েলা স্রোত (আটলান্টিক), কার্ডামম পাহাড় (দক্ষিণ ভারত), ডানডেলি অভয়ারণ্য (কর্ণাটক)।
