প্রাণীদের গমন (PYQ - সেট ২)

দিন/রাত

CLASS NO: 04

  1. দুটি হাড় মানবদেহে সংযুক্ত থাকে :

    1. জয়েন্ট
    2. লিগামেন্ট
    3. পেশী
    4. ছোট হাড়
    উত্তর: (b) লিগামেন্ট।
    ব্যাখ্যা: লিগামেন্ট হলো এক ধরনের তন্তুময় যোজক কলা যা একটি হাড়কে অন্য হাড়ের সাথে যুক্ত করে অস্থিসন্ধিকে স্থিতিশীল রাখে।

  2. জন্মের সময় একটি শিশুর শরীরে আনুমানিক কয়টি হাড় থাকে?

    1. ২০০টি হাড়
    2. ৩০০ টি হাড়
    3. 275 হাড়
    4. 250 হাড়
    উত্তর: (b) ৩০০ টি হাড়।
    ব্যাখ্যা: জন্মের সময় শিশুর দেহে প্রায় ৩০০টি অস্থি থাকে, যা বয়স বাড়ার সাথে সাথে কিছু সংযুক্ত হয়ে ২০৬টিতে পরিণত হয়।

  3. পেশীতন্ত্রের কোন অংশ হাড়কে হাড়ের সাথে সংযুক্ত করে?

    1. টেন্ডন
    2. তরুণাস্থি
    3. পেশী
    4. লিগামেন্ট
    উত্তর: (d) লিগامেন্ট।
    ব্যাখ্যা: লিগামেন্ট হলো সেই যোজক কলা যা হাড়কে হাড়ের সাথে যুক্ত করে। অন্যদিকে, টেন্ডন পেশীকে হাড়ের সাথে যুক্ত করে।

  4. মানুষের মধ্যে পেশীগুলিকে হাড়ের সাথে সংযোগকারী টিস্যু বলা হয়:

    1. টেন্ডন
    2. অ্যাক্সন
    3. ফাইবার
    4. ফিমার
    উত্তর: (a) টেন্ডন।
    ব্যাখ্যা: টেন্ডন হলো একটি কঠিন, নমনীয় এবং তন্তুময় যোজক কলা যা পেশীকে হাড়ের সাথে সংযুক্ত করে।

  5. একটি টেন্ডন তৈরি:

    1. শুধুমাত্র কোলাজেন ফাইবার
    2. ইলাস্টিক সংযোজক টিস্যু ফাইবার
    3. স্থিতিস্থাপক সংযোগকারী টিস্যু ফাইবার
    4. ইলাস্টিক
    উত্তর: (a) শুধুমাত্র কোলাজেন ফাইবার।
    ব্যাখ্যা: টেন্ডন প্রধানত ঘন, সমান্তরালভাবে সজ্জিত কোলাজেন ফাইবার দ্বারা গঠিত, যা একে প্রচণ্ড শক্তি দেয়।

  6. কশেরুকা মানুষের দেহের নিচের কোন সিস্টেমের একটি অংশ?

    1. কিডনি
    2. ফুসফুস
    3. হৃদয়
    4. স্পাইনাল কর্ড
    উত্তর: (d) স্পাইনাল কর্ড।
    ব্যাখ্যা: কশেরুকাগুলি একত্রিত হয়ে মেরুদণ্ড গঠন করে, যার মধ্যে স্পাইনাল কর্ড সুরক্ষিত থাকে।

  7. মানবদেহের নিচের কোন অংশে ম্যাসেটার পেশী অবস্থিত?

    1. হাত
    2. উরু
    3. চোয়াল
    4. বুক
    উত্তর: (c) চোয়াল।
    ব্যাখ্যা: ম্যাসেটার পেশী চোয়ালের একটি শক্তিশালী পেশী যা চিবানোর জন্য ব্যবহৃত হয়।

  8. প্রচুর পরিমাণে সাদা ফাইবার রয়েছে:

    1. লিগামেন্ট
    2. টেন্ডন
    3. তরুণাস্থি
    4. হাড়
    উত্তর: (b) টেন্ডন।
    ব্যাখ্যা: টেন্ডন শ্বেত তন্তুময় যোজক কলা দ্বারা গঠিত, যাতে প্রচুর পরিমাণে সাদা কোলাজেন ফাইবার থাকে।

  9. মানবদেহের কোন অংশে ইলিয়াম, ইসচিয়াম ও পিউবিসের মিলন ঘটলে গঠিত হয়?

    1. চোয়াল
    2. ক্রেনিয়াম
    3. নিতম্বের হাড়
    4. পা
    উত্তর: (c) নিতম্বের হাড়।
    ব্যাখ্যা: ইলিয়াম, ইসচিয়াম এবং পিউবিস—এই তিনটি হাড় একত্রিত হয়ে নিতম্বের হাড় বা হিপ বোন গঠন করে।

  10. মানুষের হাতে কয়টি হাড় থাকে?

    1. 27
    2. 21
    3. 14
    4. 17
    উত্তর: (a) 27।
    ব্যাখ্যা: মানুষের প্রতিটি হাতে মোট ২৭টি হাড় থাকে (৮টি কার্পাল, ৫টি মেটাকার্পাল, ১৪টি ফ্যালাঞ্জেস)।

  11. প্রদত্ত বিকল্প থেকে মানবদেহের হাড় এবং সংশ্লিষ্ট অংশের ভুল জোড়া নির্বাচন করুন।

    1. ম্যাক্সিলা - চোয়াল
    2. প্যাটেলা হাড় - পা
    3. কার্পাল হাড় - ঘাড়
    4. উলনা হাড় - হাত
    উত্তর: (c) কার্পাল হাড় - ঘাড়।
    ব্যাখ্যা: কার্পাল হাড়গুলি কব্জিতে অবস্থিত, ঘাড়ে নয়।

  12. নিচের কোনটি অক্ষীয় হাড় নয়?

    1. মাথার খুলি
    2. স্ক্যাপুলা
    3. মেরুদন্ডী
    4. পাঁজর
    উত্তর: (b) স্ক্যাপুলা।
    ব্যাখ্যা: মাথার খুলি, মেরুদণ্ড এবং পাঁজর হলো অক্ষীয় কঙ্কালের অংশ। স্ক্যাপুলা বা কাঁধের হাড় উপাঙ্গীয় কঙ্কালের অংশ।

  13. মানবদেহের সবচেয়ে শক্তিশালী হাড় কোনটি?

    1. গোড়ালি
    2. শিনের হাড়
    3. মেরুদন্ডী
    4. উরুর হাড়
    উত্তর: (d) উরুর হাড়।
    ব্যাখ্যা: ফিমার বা উরুর হাড় মানবদেহের দীর্ঘতম এবং সবচেয়ে শক্তিশালী হাড়।

  14. হাঁটুর ক্যাপকে বলা হয়:

    1. ফিবুলা
    2. মালুস
    3. প্যাটেলা
    4. স্ক্যাপুলা
    উত্তর: (c) প্যাটেলা।
    ব্যাখ্যা: প্যাটেলা হলো হাঁটুর সামনের দিকে অবস্থিত একটি ছোট, ত্রিকোণাকার হাড়, যা 'হাঁটুর ক্যাপ' নামে পরিচিত।

  15. মানবদেহের সবচেয়ে ছোট হাড় কোনটি?

    1. ইনকাস
    2. ম্যালেউস
    3. স্টেপস
    4. মেটাটারসাল
    উত্তর: (c) স্টেপস।
    ব্যাখ্যা: স্টেপস হলো মানবদেহের সবচেয়ে ছোট হাড়, যা মধ্যকর্ণে অবস্থিত।

  16. এই হাড়গুলোর মধ্যে কোনটি মানুষের কানের অংশ নয়?

    1. ফেমার
    2. ম্যালেউস
    3. স্টেপস
    4. ইনকাস
    উত্তর: (a) ফেমার।
    ব্যাখ্যা: ম্যালেউস, ইনকাস এবং স্টেপস—এই তিনটি হাড় মধ্যকর্ণের অংশ। ফেমার হলো উরুর হাড়।

  17. মানবদেহের সবচেয়ে বড় হাড় কোনটি?

    1. ফেমার
    2. ইনকাস
    3. ম্যালেউস
    4. স্টেপস
    উত্তর: (a) ফেমার।
    ব্যাখ্যা: ফিমার বা উরুর হাড় মানবদেহের সবচেয়ে বড় বা দীর্ঘতম হাড়।

  18. মানবদেহের দীর্ঘতম হাড় কোথায় পাওয়া যায়?

    1. বাহু
    2. পা
    3. উপরের বাহু
    4. উরু
    উত্তর: (d) উরু।
    ব্যাখ্যা: মানবদেহের দীর্ঘতম হাড় ফিমার, যা উরু বা থাই অঞ্চলে অবস্থিত।

  19. মানবদেহের সবচেয়ে ছোট হাড় কোনটি?

    1. হিউমারাস
    2. ম্যালেউস
    3. ফিমার
    4. স্টেপস
    উত্তর: (d) স্টেপস।
    ব্যাখ্যা: স্টেপস মানবদেহের সবচেয়ে ছোট হাড়, যা মধ্যকর্ণে পাওয়া যায়।

  20. কোন এসিড আমাদের পেশীতে ব্যথা সৃষ্টি করে?

    1. ল্যাকটিক এসিড
    2. সালফিউরিক এসিড
    3. অ্যাসিটিক অ্যাসিড
    4. হাইড্রোক্লোরিক অ্যাসিড
    উত্তর: (a) ল্যাকটিক এসিড।
    ব্যাখ্যা: কঠোর ব্যায়ামের সময় পেশীগুলিতে অবাত শ্বসনের ফলে ল্যাকটিক অ্যাসিড জমা হয়, যা পেশীতে ক্লান্তি এবং ব্যথার কারণ হয়।

  21. যে অস্থিসন্ধিতে একটি অস্থিকে অক্ষ করে অপর অস্থিটি ঘুরতে পারে-

    1. পিভট সন্ধি
    2. হিঞ্জ সন্ধি
    3. স্যাডল সন্ধি
    4. অচল সন্ধি
    উত্তর: (a) পিভট সন্ধি।
    ব্যাখ্যা: পিভট সন্ধিতে একটি অস্থি অন্য একটি অস্থির চারপাশে আবর্তিত হতে পারে, যেমন ঘাড়ের সন্ধি।

  22. নিচের কোনটি মানবদেহের একটি প্রধান সন্ধি?

    1. কনুই জয়েন্ট
    2. উপরের চোয়াল এবং মাথার বাকি অংশের মধ্যে জয়েন্ট
    3. জয়েন্ট যেখানে আমাদের ঘাড় মাথার সাথে মিলিত হয়
    4. কাঁধের জয়েন্ট
    উত্তর: (c) জয়েন্ট যেখানে আমাদের ঘাড় মাথার সাথে মিলিত হয়।
    ব্যাখ্যা: ঘাড় ও মাথার সংযোগকারী পিভট সন্ধি মাথাকে ঘোরানোর মতো একটি প্রধান এবং জটিল কাজ সম্পন্ন করে।

  23. মানুষের কাঁধের জয়েন্ট হল ......... জয়েন্টের উদাহরণ।

    1. স্থির
    2. প্রধান
    3. কবজা
    4. বল এবং সকেট
    উত্তর: (d) বল এবং সকেট।
    ব্যাখ্যা: কাঁধের সন্ধি একটি বল ও সকেট প্রকৃতির সন্ধি, যা প্রায় সব দিকেই নড়াচড়া করতে পারে।

  24. জয়েন্টগুলির মধ্যে উপস্থিত তরল, যা জয়েন্টগুলির চলাচলকে সহজ করে তোলে, তাকে বলা হয়:

    1. বার্সা
    2. তরুণাস্থি
    3. সাইনোভিয়াল
    4. টেন্ডন
    উত্তর: (c) সাইনোভিয়াল।
    ব্যাখ্যা: সাইনোভিয়াল সন্ধিগুলিতে সাইনোভিয়াল ফ্লুইড বা তরল থাকে, যা ঘর্ষণ কমিয়ে চলাচলকে মসৃণ করে।

  25. কোন খনিজ হাড়ের জন্য অপরিহার্য?

    1. দস্তা
    2. লোহা
    3. সালফেট
    4. ক্যালসিয়াম
    উত্তর: (d) ক্যালসিয়াম।
    ব্যাখ্যা: হাড়ের প্রধান উপাদান হলো ক্যালসিয়াম ফসফেট। ক্যালসিয়াম হাড়কে গঠন, দৃঢ়তা এবং শক্তি প্রদান করে।

  26. মানবদেহের সবচেয়ে শক্ত পদার্থ কোনটি?

    1. আঙুলের পেরেক
    2. হাঁটুর ক্যাপ
    3. মাথার খুলির হাড়
    4. দাঁতের এনামেল
    উত্তর: (d) দাঁতের এনামেল।
    ব্যাখ্যা: দাঁতের এনামেল হলো দাঁতের বাইরের শক্ত আবরণ এবং এটি মানবদেহের সবচেয়ে কঠিন পদার্থ।
